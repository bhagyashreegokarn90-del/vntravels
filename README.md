# VN Travels

A modern travel website showcasing tours and travel packages.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Interactive Navigation**: Smooth scrolling and dynamic navigation
- **Destination Showcase**: Beautiful grid layout of popular Vietnam destinations
- **Travel Packages**: Multiple tour packages with pricing and features
- **Contact Form**: Easy-to-use contact form for inquiries
- **Modern Styling**: Clean and professional design with gradient backgrounds

## Files Included

- `index.html` - Main HTML file with all page content
- `styles.css` - Complete styling and responsive design
- `script.js` - Interactive functionality and animations
- `README.md` - Documentation

## Sections

1. **Header**: Navigation bar with links to all sections
2. **Hero**: Eye-catching welcome section
3. **Destinations**: Popular Vietnam destinations (Hanoi, Ha Long Bay, Ho Chi Minh City, Da Nang)
4. **Packages**: Travel package options with pricing
5. **About**: Information about VN Travels
6. **Contact**: Contact form for customer inquiries
7. **Footer**: Copyright information

## Getting Started

1. Clone or download this repository
2. Open `index.html` in a web browser
3. All images are currently using placeholder paths - update image URLs as needed

## Color Scheme

- Primary Color: Vietnamese Red (#c41e3a)
- Secondary Color: Vietnamese Gold (#ffde00)
- Dark: #333
- Light: #f4f4f4

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## Customization

To customize the website:

1. Update the destination cards with real images and links
2. Add your own travel packages and pricing
3. Modify colors in the `:root` CSS variables
4. Update contact form to connect to your backend service
